# Job Ops User Guide

> Public-facing guide for pilot participants. Learn how to use Job Ops for your job search.

Last updated: April 29, 2026

## 1. Welcome to Job Ops

Job Ops helps you manage a smart, organized job search:

- Track jobs, applications, and AI-generated drafts
- Build a base resume and keep it versioned
- Tailor resumes and cover letters for specific jobs
- Use AI parsing to extract requirements from job descriptions
- Track every application in one place
- Get AI-assisted freelance proposals

**Core principles:** Quality over quantity. Truthfulness over embellishment. Human review before use.

## 2. Getting Started

### Create Your Account
Visit `/register` with a valid invite code. Verify your email and sign in.

### Build Your Base Resume
Go to the Base Resume page. Paste or type your master resume. Every edit creates an immutable version — you can always restore earlier versions.

### Set Up Your Claims
The Claims Ledger holds verified facts about your experience. Add claims one at a time. These are used when tailoring resumes — AI will only use facts you've approved.

## 3. Job Pipeline

1. **Ingest a job** — Paste a job description URL or text
2. **Run the parser** — AI extracts required skills, nice-to-haves, responsibilities, and keywords
3. **Review and edit** — Verify AI output before continuing
4. **Attach a role profile** — Group similar jobs with shared claims
5. **Select claims** — Pick which experience facts to highlight
6. **Tailor resume** — Generate a job-specific resume using only approved claims
7. **Draft cover letter** — AI drafts attributed paragraphs
8. **Approve or reject** — Human approval gate before using any output
9. **Submission prep** — Move to assisted apply for file generation

## 4. Resume Versions

Each tailored resume is a separate version. You can:
- View, approve, or reject any version
- Compare multiple AI model outputs
- Export as DOCX
- Track which version was used for each application

## 5. Cover Letters

Cover letters are drafted with paragraph-level claim attribution:
- Each paragraph cites the claims it's based on
- You can approve or reject individual versions
- Export as DOCX when ready

## 6. Applications

The Applications page tracks every application:
- Job, resume version, cover letter version
- Status (applied, interview, offer, rejected)
- Feedback signals for AI learning
- Application date tracking

## 7. Assisted Apply

The Assisted Apply feature helps you prepare application materials:
- Select platform (Greenhouse, Lever, Workday)
- Set human checkpoint preferences
- Generate ready-to-use files

**Important:** Job Ops does NOT auto-submit applications. Every submission requires your manual review and action.

## 8. Freelance Copilot

For freelance/contract work:
- Create project profiles
- Draft proposals using AI
- Track proposal status
- Export for manual submission on Upwork or similar platforms

## 9. AI Review & Learning

Job Ops uses AI to assist — not replace — your judgment:
- Prompt versions are tracked for auditability
- AI runs are logged with inputs/outputs
- You can provide feedback on AI quality
- The system learns from application outcomes

## 10. Account & Settings

- Change password and email under Account settings
- Enable Two-Factor Authentication (2FA) for extra security
- View your usage limits under Account

## 11. FAQ

**Can I auto-apply to jobs?**
No. Job Ops is designed for thoughtful, human-reviewed applications. Auto-submit is intentionally not supported.

**How many applications can I create?**
Pilot users have a weekly AI request quota (default 5/week). View your usage in Account settings.

**Where is my data stored?**
Your data is stored in a secure PostgreSQL database hosted on Neon and managed by L&D PRO.

**How do I report a bug or give feedback?**
Use the feedback widget (bottom-right corner) or email support.

**Can I request more AI requests?**
Pilot users can request limit increases through the feedback widget or by contacting support.
